Website Redesign

Local Audio Store

Project Type

UI/UX Design, Web Development

Role

Designer

Overview

A class assignment where needed to take a local business page and redesign the UI. A minimum of 3 pages were needed to be redesigned. The design must then be developed using html5 and css3.

Discovery

The audio store is a city staple and sells several different types of products as well as home services like home security and home theatre setups.

Sitemap

I reorganized their sitemap to prioritize their products first as that was their primary selling point.

Concept

I wanted draw from the grandeur of their home theatre setups. So the hero pages, I decided, needed to evoke that same spirit using a modernist sensibility.

Visual Design

I wanted draw from the grandeur of their home theatre setups. So the hero pages, I decided, needed to evoke that same spirit using a modernist sensibility. My colour palette holds onto a sophistication, using dark tones but button UI elements would take on a brighter toned pink. Sections like the featured cards have an offset image to give a more contemporary feel.
